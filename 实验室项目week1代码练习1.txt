考试一共有3门科目：数学，语文，英语，小乐乐的班主任决定给没有通过考核的同学家长开一次家长会，考核的标准是三科平均分不低于60分
输入描述:
一行，输入三个整数（表示小乐乐的数学、语文、英语的成绩），用空格分隔。
输出描述:
一行，如果小乐乐会被请家长则输出“YES”，否则输出“NO”。


#include<stdio.h>
int main()
{
  int a,b,c,s;
  scanf("%d %d %d",&a,&b,&c);
  s=(a+b+c)/3;
  if(s>=60)
  printf("NO");
  else
  printf("YES");
 return 0;
}
